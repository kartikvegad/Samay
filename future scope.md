# Future Scope

The Samay dashboard has a solid foundation, but there are numerous avenues for future improvements and features:

## Planned Features

1. **User Accounts & Preferences**
   - Enable users to create accounts to save their favorite cities and dashboard configurations across devices.

2. **Expanded City Support & Search**
   - Implement a dynamic search bar with auto-complete to easily add any city in the world to the dashboard.
   - Fetch real-time timezone offsets using a third-party geolocation or timezone API.

3. **Alarms and Notifications**
   - Allow users to set alarms or meeting reminders tied to specific city timezones.
   - Browser notification support for cross-timezone events.

4. **Interactive Timeline/Meeting Planner**
   - A visual slider that lets users drag the current time forward/backward to effortlessly calculate business overlap hours between multiple selected cities.

5. **Weather Integration**
   - Display a small, clean icon and current temperature alongside the time for each city card.

6. **Customizable Layouts**
   - Allow dragging and dropping of city cards to reorder the dashboard according to user priority.

7. **Localization**
   - Support for multiple languages, translating date and time formats based on the user's local locale settings.
